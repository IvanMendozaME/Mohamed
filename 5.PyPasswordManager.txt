		Password Manager

The purpose of this project is get a customize software to save the passwords.

This program consist in 3 inputs: website, Email/Username and the password. 
If you do not have a password crated, there is a button to create a random password
using letters, numbers and symbols. 

At the moment to add with all the inputs completed. The information is saved 
in a text file named data, there is all the information that the user have 
saved.
